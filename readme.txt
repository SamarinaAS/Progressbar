Программа, позволяющая загрузить некоторый файл из интернета, вставив на него ссылку. При загрузке файла показывается полоска загрузки (progressbar) и процент загрузки. Есть возможность отменить загрузку, нажав кноку "cancel". Графический нтерфейс выполнен с помощью библиотеки wxwidgets, процесс загрузки файла с помощью библиотеки curl.
Собиралось-запускалось из командной строки ubuntu с помощью команды:
g++ -o progressbar progressbar.cpp `wx-config --cxxflags` `wx-config --libs` -lcurl -pthread
